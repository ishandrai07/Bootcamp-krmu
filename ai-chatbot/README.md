# AI Chatbot Demo (React + Node.js + Mistral AI)

A simple, workable demo of an AI chatbot built using **React**, **Node.js/Express**, and the **Mistral AI API**.

### Features
- 🚀 Hero section with a large interactive chat box
- 💬 Click to open a floating chat window
- ⌨️ Type messages or use voice input (Browser Web Speech API)
- 🤖 AI-powered responses from **Mistral AI** via a secure Node.js backend
- 🔊 Optional text-to-speech for AI responses
- 💾 Conversation history maintained for contextual responses

> This project is intended for learning and demonstration purposes and is not production-ready.

---

# 1. Get a Mistral AI API Key

Create a free API key from the Mistral AI Platform:

https://console.mistral.ai/

After signing in:

- Go to **API Keys**
- Generate a new API key
- Copy the key

---

# 2. Set Up the Server

```bash
cd server
npm install
cp .env.example .env
```

Open `.env` and add your API key:

```env
MISTRAL_API_KEY=your_real_api_key_here
PORT=5000
```

Start the backend server:

```bash
npm run dev
```

You should see:

```text
Server running on http://localhost:5000
```

---

# 3. Set Up the Client

Open another terminal:

```bash
cd client
npm install
npm run dev
```

Open the URL displayed by Vite (usually):

```
http://localhost:5173
```

---

# 4. Try the Chatbot

1. Open the application.
2. You'll see a hero section with a large **"Talk to the AI Assistant"** box.
3. Click the box to open the chatbot.
4. Type a message and press **Enter** or click **Send**.
5. Or click the **🎤 microphone** button to speak.
6. The response is generated by **Mistral AI** and displayed in the chat.
7. If **"Read replies aloud"** is enabled, the browser will read the response using Text-to-Speech.

# Notes for Demonstration

- 🎤 Voice input uses the browser's built-in **SpeechRecognition API**.
- It works best in **Google Chrome** and **Microsoft Edge**.
- Firefox and Safari have limited support.
- The **Mistral API key** is stored only on the backend (`server/.env`) and is **never exposed to the frontend**, following security best practices.
- Conversation history is stored in React state and sent with each request, allowing the AI to maintain context throughout the conversation.

# Common Issues

### "MISTRAL_API_KEY is not set"

You forgot to create the `.env` file or add your API key.

---

### Couldn't connect to the server

- Ensure the backend is running.
- Verify it is listening on **port 5000**.
- Check the frontend API URL.

---

### AI isn't responding

- Verify your Mistral API key is valid.
- Check your internet connection.
- Confirm the selected model name exists.

---

### Microphone button doesn't work

Your browser either:

- doesn't support the SpeechRecognition API, or
- microphone permission was denied.

Allow microphone access and refresh the page.

---
